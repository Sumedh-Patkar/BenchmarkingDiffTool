FACTS v1.6 (Latest version)

Major features - Timeline Graph

Timeline Graphs
  • Users can generate timeline graph for a benchmark on allRuns page, shown on a Modal
  • The graph has the following filters
    • Result Type
    • SMT
    • SKU
    • yParameter
    • xParameter
  • The graph shows month wise improvement in the numbers of that benchmark
Other changes
  • New and beautiful colour scheme for bar graphs. Related SKUs have similar shade colours. Darker colours for newer generations
  • Dashboard Graphs have Legend skus according to sku_definition.ini. Applied custom sorting on dataframe
  • Changed styles for reports page. Benchmark names are now vertically scrollable
  • Focus is moved to search bar on clicking filter by benchmark/label in reports page
Bug Fixes
  • Added exception handling in CPU Utilization graphs. Empty graphs shown where files do not exist
  • Compare page shows result type correctly now

FACTS v1.5

Major features - Shareable Dashboard links and UI Changes

Shareable links
  • The graphs dashboard on all-runs page has the following filters
    • xParameter
    • yParameter
    • Input Filters List
    • Result type
      • Changing any one of them changes the URL of the page and redraws the graph

  • This URL can then be shared directly with non technical users, and the graphs on the dashboard will be drawn according to the filters in the URL
Major UI Changes
  • All Runs page style has been drastically improved
  • All input filters are hidden by default. Clicking on 'view input filters' shows the element dropdown
  • Only result type filter is show
Other changes
  • Added sorting arrow to jQuery Datatables
Bug Fixes
  • Removed unused old css properties
  • Removed unused package 'joblib' from imports and requirements.txt

FACTS v1.3

Performance Improvements
  • SKU comparison graphs on all Runs page used to take around 20 to 80 seconds to load.

  • Main page graph used to take around 200-250 seconds despite applied parallelization using multiprocessing

    (~25x Performance improvement)
Other changes

  • Focus on search textbox when 'Search benchmark' dropdown clicked
  • Added links on test details page. Buttons are now outside the results div, fixed
  • Add Result Type filter all runs page
Bug Fixes

  • Clearing live search textbox (top left) on refresh
  • Removed unnecessary form element from 'Search Benchmark' which on pressing enter would get submitted and give error
  • Fixed bug in compare tests. Removed POST request as an accepted method in diff

FACTS v1.2

Major features - Reports Generation and New Graphs

Reports Generation

Users can generate custom excel sheets in three steps

  1. Select Labels or Benchmarks
  2. Select Best results / Top 5 / All results
  3. Advanced filters like SKU, Scaling, Kernel, OS, etc.

New Graphs

  • iostat graphs

    Multiline graph for iostat is added to CPU Utilization graphs having kB_read/s and kB_write/s on y-axis

  • Scaling Graphs

    These are the default graphs on the left side of All Runs page

Other additions
  • Download all graphs at once in CPU Utilization graphs
  • Search benchmark names in the "Go to Benchmark" dropdown. No need of scrolling all the way down
  • Responsive loading circle for Graphs, which disappears after the graph has loaded. This will assure user that graph is being processed.

Performance improvements and Bug Fixes
  • Some MYSQL queries have been optimized, Caching has been added for reports generation
  • Several bugs have been fixed. See github commit history and comments for more details
